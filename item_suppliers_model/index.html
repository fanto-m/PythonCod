    class ItemSuppliersModel¶

    Модель данных для управления списком поставщиков, связанных с конкретным товаром.

    Наследуется от QAbstractListModel. Предоставляет данные о поставщиках для QML-интерфейса, включая их идентификатор, имя, компанию, email, телефон и веб-сайт. Поддерживает загрузку данных из базы данных через DatabaseManager и обновление данных при изменении артикула товара.

    Args: article (str, optional): Артикул товара, для которого загружаются поставщики. По умолчанию пустая строка.

    def load(self)

    Загрузка данных поставщиков для текущего артикула.

    Сбрасывает модель, получает идентификаторы поставщиков для текущего артикула из базы данных и загружает полные данные для соответствующих поставщиков.
